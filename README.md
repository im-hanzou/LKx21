</div>

> [!TIP]

## Options

| Options | Name  | Description  |
| :-----: | ----- | ------------ |
| -t      | Title | Movie title  |
| -l      | Limit | Search limit |


## Usage
```
python dl.py -t <title>
```
**Example :**
```
python dl.py -t sherlock
```
```
python dl.py -t "Incantation 2022" -l 2
```
